# 🍄 Cumbucha Telegram Bot

Телеграм-бот для отслеживания стадий ферментации камбучи. Отправляет уведомления в канал и ведёт один закреплённый пост с расписанием.

## Возможности

- 🌱 Режим **выращивания SCOBY** — уведомления на дни 3, 7, 10, 14, 21
- ⚗️ Режим **первой ферментации** — уведомления на дни 2, 5, 7, 10
- 🫧 Режим **второй ферментации** (газирование) — уведомления на дни 1, 2, 3
- 📌 Один закреплённый пост в канале — обновляется автоматически
- 💾 Состояние сохраняется в `state.json` — переживёт перезапуск

## Быстрый старт

### 1. Создай бота в Telegram

1. Напиши [@BotFather](https://t.me/BotFather) команду `/newbot`
2. Следуй инструкциям, получи **токен** (выглядит как `123456:ABCdef...`)
3. Добавь бота **администратором** в свой канал

### 2. Узнай ID канала

Перешли любое сообщение из канала боту [@userinfobot](https://t.me/userinfobot) — он покажет `id` канала.  
Формат: `-100xxxxxxxxxx` (для публичных можно использовать `@username`).

### 3. Склонируй репозиторий

```bash
git clone https://github.com/YOUR_USERNAME/cumbucha-bot.git
cd cumbucha-bot
```

### 4. Установи зависимости

```bash
pip install -r requirements.txt
```

### 5. Задай переменные окружения

```bash
export BOT_TOKEN="123456:ABCdef..."
export CHANNEL_ID="@mychannel"   # или -100xxxxxxxxxx
```

### 6. Запусти

```bash
python bot.py
```

При первом запуске бот автоматически создаст варку SCOBY с сегодняшней датой и опубликует закреплённый пост.

## Команды бота

Пиши команды боту **в личку** (не в канал):

| Команда | Действие |
|---|---|
| `/start` | Показать справку |
| `/newscoby` | Начать выращивание SCOBY (сегодня) |
| `/newbrew` | Начать первую ферментацию |
| `/newf2` | Начать вторую ферментацию (газирование) |
| `/status` | Показать текущее состояние варок |
| `/clear` | Удалить все варки |

## Деплой на сервер (Railway / VPS)

### Railway (бесплатно, рекомендую)

1. Зарегистрируйся на [railway.app](https://railway.app)
2. Создай новый проект → Deploy from GitHub
3. В настройках добавь переменные окружения:
   - `BOT_TOKEN` = твой токен
   - `CHANNEL_ID` = ID канала
4. Railway сам запустит `python bot.py`

### VPS / systemd

Создай файл `/etc/systemd/system/cumbucha-bot.service`:

```ini
[Unit]
Description=Cumbucha Telegram Bot
After=network.target

[Service]
WorkingDirectory=/path/to/cumbucha-bot
ExecStart=/usr/bin/python3 bot.py
Restart=always
Environment=BOT_TOKEN=your_token_here
Environment=CHANNEL_ID=your_channel_id

[Install]
WantedBy=multi-user.target
```

```bash
sudo systemctl enable cumbucha-bot
sudo systemctl start cumbucha-bot
sudo systemctl status cumbucha-bot
```

## Расписание уведомлений

### 🌱 Выращивание SCOBY

| День | Событие |
|---|---|
| 3 | Первый осмотр — появление «облака» на поверхности |
| 7 | SCOBY растёт — проверка слоя (1–3 мм) |
| 10 | Проверка толщины — должно быть 3–5 мм |
| 14 | SCOBY формируется — 5–8 мм |
| 21 | SCOBY готов к первой полноценной варке! |

### ⚗️ Первая ферментация

| День | Событие |
|---|---|
| 2 | Проверка запаха |
| 5 | Первая проба вкуса |
| 7 | Основная проверка готовности |
| 10 | Финальный дедлайн — разливай! |

### 🫧 Вторая ферментация (газирование)

| День | Событие |
|---|---|
| 1 | Проверка бутылок |
| 2 | Проверка давления — готово? |
| 3 | Скорее всего готово — в холодильник! |
